# Histock::Simplefilter

[HiStock嗨投資理財社群](https://histock.tw/) filter by Ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'histock-simplefilter'
```

And then execute:

```
$ bundle install
```

Or install it yourself as:

```
$ gem install histock-simplefilter
```
